Test Pages

The test pages are used for quality assurance of SkipTo.js features. For example what if a page does not have any landarks, does SkpTo.js menu provide information indicating this condition.

Test Description
No landmarks No landmarks on page
No headings No headings on page
Uses Role Heading Heading role used on page and empty H2, h3 and H4 elements
Configuration Object Uses the SkipToConfig javascript object for configuring SkipTo.js
Input controls Shortcut keys is disabled when focus is a text input.
Custom Elements with Slotted Content Uses custom elements to render headings and landmarks.
Header Size Testing if the dimensional size (e.g. height and width) of headings effects which headings screen readers will include in lists and header navigation

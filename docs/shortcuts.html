Shortcut Keys

The browser extensions and bookmarklet versions of SkipTo.js provide powerful keyboard navigation to landmark regions and the headings of a web page. Structural navigation to regions and headings have long been a part of W3C User Agent Accessibility Guidelines 1.0 and W3C User Agent Accessibility Guidelines 2.0 requirements. Screen readers for the blind have provided region and heading navigation features for over 15 years. Now keyboard support to landmark regions and headings are available through free browser extensions.
